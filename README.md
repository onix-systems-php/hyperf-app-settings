# Hyperf-app-settings component

Includes the following classes:

- Exception:
  - BusinessException.
- Event:
  - Action. 
- Constants:
  - AppSettings;
  - ErrorCode;
  - Time.
- DTO:
  - UpdateAppSettingDTO.
- Model:
  - AppSetting;
  - User.
- Repository:
  - AppSettingsRepository.
- Service:
  - AppSettingsService.
  - UpdateAppSettingsService.

Install:
```shell script
composer require onix-systems-php/hyperf-app-settings
```
